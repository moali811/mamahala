# Mama Hala Consulting — Keyword Strategy (EN/AR)

Living document. Update quarterly based on Google Search Console performance data.

**Target markets**: Canada (Ottawa + national) and UAE (Dubai + GCC wide, Arabic-speaking audiences across MENA).

**Brand positioning**: Yale-trained, bilingual (EN/AR), culturally fluent counseling for individuals, couples, and families. Online and in-person. Warm, professional, faith-and-culture-respectful.

---

## Search Intent Framework

Every target keyword should map to one of these intents:

| Intent | Page type | Example |
|---|---|---|
| **Transactional** (ready to book) | Service detail, booking | "book family counselor Ottawa" |
| **Commercial** (comparing providers) | Service category, about, location pages | "best Arabic counselor Dubai" |
| **Informational** (researching a problem) | Blog, FAQ, quizzes | "signs my teenager needs therapy" |
| **Navigational** (looking for us by name) | Home, about | "Dr Hala Ali counselor" |

## Geo-Modifier Strategy

Mama Hala serves two distinct markets. Use geo-keywords on:
- Home page H1/meta (subtly: "serving Ottawa and Dubai")
- About page (locations section)
- Dedicated location landing pages (see recommendation below)

**High-priority geo-keywords**:

| EN | AR |
|---|---|
| counselor Ottawa | مستشار أوتاوا |
| family therapist Canada | معالج أسري كندا |
| counselor Dubai | مستشار دبي |
| Arabic counselor UAE | مستشار عربي الإمارات |
| online counselor Canada Dubai | استشارات أونلاين كندا دبي |
| bilingual therapist Canada | معالج ثنائي اللغة كندا |

**Recommendation: build 2 location pages** (Phase 4.2 in main SEO plan):
- `/en/counseling-ottawa` + `/ar/استشارات-أوتاوا` (or kept English: `/ar/counseling-ottawa`)
- `/en/counseling-dubai` + `/ar/استشارات-دبي`

Each should include: H1 with location + primary keyword, local address + hours, local testimonials, 3-5 location-specific FAQs, embedded map, `LocalBusiness` JSON-LD.

## Internal Linking Rules

Apply consistently as posts and services are added:

1. **Blog post body** → link to 1-2 relevant services (natural anchor text, not "click here")
2. **Blog post footer** → "Related services" card linking to matching service category
3. **Service detail page** → link to 2-3 related blog posts in sidebar/footer
4. **FAQ answer** → if answer mentions a service, link to it
5. **Category page** → link to recent blog posts in that category
6. **Home page** → rotate 3 featured blog posts tied to current seasonal focus

## Tracking & Measurement

Once GA4 + GSC are wired up (Phase 1.6 of main plan):

**Monthly SEO review metrics**:
- Impressions by landing page (EN vs AR split)
- Top queries by CTR (both locales)
- Average position for primary keywords (tracked per category)
- Indexation coverage (GSC)
- Arabic query discovery (what terms Arabic users actually search)

**Quarterly adjustments**:
- Rebalance blog calendar based on which posts rank well
- Promote high-performer service pages with more internal links
- Prune or update underperforming content
